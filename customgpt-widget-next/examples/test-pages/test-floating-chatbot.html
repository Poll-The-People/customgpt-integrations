Floating Widget Integration

This integration style provides a floating chat widget that appears as a button in the bottom-right corner of your screen. It's the same pattern used by popular chat platforms.

Key advantages of this approach:

Always Accessible

  • Visible on every page
  • Doesn't interrupt content
  • Familiar chat interface
  • Easy to open and close

Responsive Design

  • Full-screen on mobile
  • Widget on desktop
  • Smooth animations
  • Touch-friendly controls

Quick Integration

  • Single script tag
  • No HTML changes needed
  • Works on any page
  • Fully customizable

How to Integrate

Adding the floating chatbot to your website is incredibly simple. Just include one script configuration in your HTML - no complex setup required.

<!-- Add this script to your HTML --> <script> window.customGPTConfig = { serverUrl: 'https://your-project.vercel.app', // Your deployed widget URL position: 'bottom-right', theme: 'dark', initialMode: 'chat' }; </script> <script src="https://your-project.vercel.app/widget.js" defer></script> <!-- That's it! The widget will appear automatically -->

The script creates:

  • A floating button in the bottom-right corner
  • A chat widget that slides in smoothly
  • Voice and text interaction capabilities
  • Mobile-responsive behavior
  • Keyboard shortcuts (ESC to close)

Deployment URLs

After deploying to Vercel or Railway, replace the placeholder URL with your actual deployment:

  • Vercel: https://your-project.vercel.app
  • Railway: https://your-project.railway.app
  • Custom Domain: https://yourdomain.com

Customization Options

You can customize the appearance and behavior through the config object:

window.customGPTConfig = { serverUrl: 'https://your-project.vercel.app', position: 'bottom-right', // bottom-right, bottom-left theme: 'dark', // dark, light initialMode: 'chat', // chat, voice primaryColor: '#8b5cf6', // Your brand color width: '400px', // Widget width height: '600px' // Widget height };

Features in Action

Try these interactions with the floating widget:

  • Click the floating button - Opens the widget with a smooth slide-in animation
  • Activate Voice Mode - Click the microphone in the widget to start voice interaction
  • Press ESC - Quickly close the widget using keyboard shortcut
  • Click X button - Close the widget with the header button
  • Resize window - See how it adapts to mobile screens
