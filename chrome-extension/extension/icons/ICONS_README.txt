ICON PLACEHOLDER FILES

- icon16.png (16x16px)
- icon48.png (48x48px)
- icon128.png (128x128px)

You can create icons at:
- https://www.canva.com (free)
- https://www.figma.com (free)
- Or use any image editor

Tip: Use your brand colors and a simple, recognizable symbol
